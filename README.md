# Health Chatbot: AI-Powered Personalized Health Assistant
### Final Year Project Phase - 2
An AI-powered personalized health assistant evolving into a comprehensive Health Super App with on-device processing capabilities.

## Core Features

**On-Device AI Processing**
- Gemma 2B LLM integration for local processing
- Enhanced RAG system for health domain
- Multi-model AI architecture for improved diagnostics
- Zero data sharing architecture

**Health Monitoring**
- Smart diagnostics with personalization
- Health records management
- Real-time vitals monitoring
- AI-powered fitness and nutrition recommendations

**Platform Support**
- Standalone Wear OS application
- Cross-device synchronization
- Offline functionality
- Privacy-focused processing

## Technical Architecture
- Frontend: Streamlit
- Backend: Flask
- Database: ChromaDB vector storage
- AI Models: 
  - Gemma 2B LLM (on-device)
  - Enhanced RAG system
  - Local processing pipeline

## Premium Features
- Cloud LLM access
- Advanced analytics dashboard
- Priority feature access
- Enhanced diagnostic capabilities
